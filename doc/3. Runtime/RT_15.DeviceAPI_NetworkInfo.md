### 3) Module Attribute

- navigator.connection.type;

	- Description : 현재 네트워그 연결 타입을 얻는 기능 
	- W3C Spec : Working Draft in Deviceapi Working Group
	- Type : attribute 
	- Full Example :

			var type = navigator.connection.type
